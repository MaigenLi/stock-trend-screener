## 🎯 快速开始

### 方法1：使用全市场扫描工具（推荐）
```bash

# 选择参数组合：
#   1. 严格参数 (筛选比例1-3%)
#   2. 中等参数 (筛选比例5-10%) ← 推荐
#   3. 宽松参数 (筛选比例10-20%)

```

### 方法2：直接使用主筛选器
```bash
# 扫描全部股票，使用中等参数
python trend_volume_screener.py --all \
  --price-above ma5 \
  --min-three-day 3 \
  --max-three-day 30 \
  --min-volume-ratio 1.0 \
  --min-score 70 \
  --workers 20
```

### 方法3：使用简单筛选器（快速测试）

## 📊 结果解读

### 输出内容说明
```
📈 找到 156 只符合条件的股票，显示前 20 只:

 1. sh600250 未知 (评分:107.0)
    价格:   12.47 (+3.40%)
    三天: +6.04% (3/3上涨连续)
    量比: 1.64 连续放量
    趋势: 60% (MA5↑ MA10↑ MA5>MA10 MA10>MA20)
```

### 关键指标
- **评分**: 综合评分，越高越好（0-120分）
- **三天涨幅**: 最近三天累计涨幅
- **量比**: 成交量放大程度（>1.0表示放量）
- **趋势强度**: 最近10日上涨天数比例
- **趋势描述**: MA均线排列状态

## 🔧 参数调节技巧

### 根据市场环境调节
- **强势市场** → 提高趋势和量能要求
- **震荡市场** → 使用中等参数
- **弱势市场** → 放宽条件，关注超跌反弹

### 根据筛选结果调节
- **筛选结果太少** (<20只) → 放宽条件
- **筛选结果太多** (>200只) → 收紧条件
- **筛选比例理想** (5-20%) → 参数合理

## ⚠️ 注意事项

### 数据时效性
- 通达信数据有1天延迟
- 实时行情需要额外API
- 建议结合其他数据源验证

### 风险控制
- 技术分析需结合基本面
- 设置止损位（建议5-8%）
- 分散投资（3-5只股票）

### 性能考虑
- 全市场扫描需要20-40分钟
- 建议使用20个并行线程
- 可分批处理避免内存问题

## 📞 故障排除

### 常见问题
1. **找不到股票数据** → 检查通达信数据目录
2. **运行速度慢** → 减少并行线程数
3. **筛选结果异常** → 检查参数设置
