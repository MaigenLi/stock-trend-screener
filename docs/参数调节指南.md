# 趋势筛选参数调节指南

## 📊 参数分类说明

### 1. 趋势参数（控制趋势严格度）
- **`require_price_above_ma5`**: 价格在MA5之上（基础条件）
- **`require_price_above_ma10`**: 价格在MA10之上（更严格）
- **`require_ma5_above_ma10`**: MA5>MA10（短期趋势向上）
- **`require_ma10_above_ma20`**: MA10>MA20（中期趋势向上，最严格）
- **`min_trend_strength`**: 趋势强度（0.0-1.0，最近10日上涨天数比例）

### 2. 三天表现参数（控制短期表现）
- **`min_three_day_change`**: 最小三天涨幅（%）
- **`max_three_day_change`**: 最大三天涨幅（%）
- **`min_up_days`**: 最小上涨天数（1-3）
- **`require_consecutive_up`**: 要求连续三天上涨（最严格）

### 3. 量能参数（控制量能要求）
- **`min_volume_ratio`**: 最小平均量比
- **`require_consecutive_volume`**: 要求连续三天放量（最严格）

### 4. 风险参数（控制风险）
- **`max_ten_day_change`**: 最大十日涨幅（%）

### 5. 其他参数
- **`min_price`**: 最小价格（元）
- **`max_price`**: 最大价格（元）
- **`min_score`**: 最小综合评分（0-100）

## 🔧 参数调节建议

### 根据市场环境调节
1. **强势市场（牛市）**:
   - 提高趋势要求（`require_ma10_above_ma20=True`）
   - 提高量能要求（`min_volume_ratio=1.2`）
   - 提高涨幅要求（`min_three_day_change=5.0`）

2. **震荡市场**:
   - 使用中等参数
   - 关注趋势强度（`min_trend_strength=0.6`）
   - 适当放宽量能要求

3. **弱势市场（熊市）**:
   - 放宽趋势要求
   - 降低量能要求
   - 提高风险容忍度

### 根据投资风格调节
1. **价值投资者**:
   - 关注长期趋势（`require_ma10_above_ma20=True`）
   - 严格风险控制（`max_ten_day_change=30.0`）
   - 不追求短期暴涨

2. **趋势交易者**:
   - 关注短期趋势（`require_ma5_above_ma10=True`）
   - 要求放量上涨（`min_volume_ratio=1.2`）
   - 接受较高风险

3. **短线投机者**:
   - 放宽趋势要求
   - 关注量能变化
   - 接受较高波动

## 📈 效果评估指标

### 1. 筛选比例
- **理想范围**: 5-20%
- **<5%**: 条件可能过严
- **>20%**: 条件可能过宽

### 2. 平均评分
- **理想范围**: 70-85分
- **<70分**: 质量可能不高
- **>85分**: 可能错过机会

### 3. 特征统计
- **平均三天涨幅**: 3-10%为佳
- **平均量比**: 1.0-1.5为佳
- **趋势强度**: ≥60%为佳

## 🚀 快速调节方法

### 方法1：逐步放宽法
1. 从严格参数开始
2. 如果筛选结果太少（<5只）
3. 逐步放宽最难满足的条件
4. 直到筛选比例达到5-20%

### 方法2：核心条件法
1. 确定2-3个核心条件（如：价格在MA5之上、放量上涨）
2. 其他条件作为加分项
3. 通过评分系统控制质量

### 方法3：市场适应法
1. 观察市场整体趋势
2. 根据市场强度调节参数
3. 定期（每周）回顾和调整
