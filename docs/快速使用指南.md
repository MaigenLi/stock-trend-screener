# stock_trend 快速使用指南

## 快速开始

### 方式1：全市场扫描（推荐）

```bash
cd ~/.openclaw/workspace/stock_trend
python full_scan_auto.py
```

### 方式2：快速测试少量股票

```bash
python quick_test.py
```

### 方式3：指定日期扫描

```bash
python full_scan_auto.py --date 2026-04-02
```

## 核心文件

| 文件 | 说明 |
|------|------|
| `core/trend_volume_screener_ultimate.py` | 趋势放量筛选器（终极版） |
| `core/final_full_market_scan.py` | 全市场扫描实现 |
| `full_scan_auto.py` | 自动扫描入口（推荐使用） |
| `quick_test.py` | 快速功能测试 |

## 常用命令

```bash
# 查看帮助
python full_scan_auto.py --help

# 指定日期扫描全市场
python full_scan_auto.py --date 2026-04-02

# 指定输出文件
python full_scan_auto.py --date 2026-04-02 --output results.txt

# 调整并行线程数
python full_scan_auto.py --workers 30
```

## 筛选逻辑

筛选器对每只股票检查以下条件：

**必须满足（否则一票否决）：**
- 价格在60日均线上方
- 三天连续上涨（涨幅在3%~30%）
- 成交量放大（量比 ≥ 1.0）

**加分项（计算综合评分）：**
- 趋势强度（MA多头排列）
- 放量持续性
- 板块热度

## 结果解读

典型输出：

```
📈 找到 23 只符合条件的股票，显示前 20 只:
 1. sh600519  贵州茅台  评分: 95.0
    价格: 1680.00  涨幅: +3.2%
    三天: +8.5%  量比: 1.8
    趋势: MA5↑ MA10↑ MA10>MA20
```

- **评分**：综合评分，80+ 为优质候选
- **三天涨幅**：最近3天累计涨幅
- **量比**：当日成交量 / 20日均量，>1.0 表示放量

## 注意事项

- 通达信离线数据通常有1天延迟
- 盘中建议配合 `stock_common` 获取实时数据
- 筛选结果仅作参考，需结合其他分析
