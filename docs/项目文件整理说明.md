# 📊 Stock Trend 项目文件整理说明

## 📁 项目概述
本项目是一个完整的股票趋势筛选系统，专注于识别处于上升趋势中、最近三天放量上涨的股票。系统支持参数可调节，适应不同市场环境。

## 📋 **使用指南**

### 1. **日常使用流程**
```bash
# 快速测试参数
python quick_full_test.py

# 全市场扫描（交互式）
python final_full_market_scan.py

# 直接使用主筛选器
python trend_volume_screener.py --all --price-above ma5 --min-three-day 3 --max-three-day 30 --min-volume-ratio 1.0 --min-score 70
```

### 2. **参数调节参考**
- **严格参数**: 筛选比例1-3%，质量最高
- **中等参数**: 筛选比例5-10%，平衡质量数量（推荐）
- **宽松参数**: 筛选比例10-20%，机会最多

### 3. **查看结果**
```bash
# 查看最新结果
ls -la results/latest/

# 查看具体文件
cat results/latest/trend_volume_stocks_*.txt | head -50
```
